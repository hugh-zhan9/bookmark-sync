# Bookmark Sync 需求说明（整理版）

我想构建一个软件，主要作用是做浏览器书签同步与管理。

## 核心需求

1. 支持浏览器书签接入：
- 同步浏览器当前已有书签
- 同步后续新增/变更书签

2. 提供专业桌面端界面：
- 书签列表展示
- 文件夹管理
- 标签管理
- 搜索与过滤

3. 支持多端增量同步：
- 基于 Git 仓库做事件级增量同步
- 同一个 URL 在系统中仅保留一条记录（去重）

## 当前范围补充

- 浏览器到 App：支持手动导入、启动自动导入、定时自动导入
- App 到 Git：支持启动 Pull、定时同步、关闭 Push
- 同步配置可在设置页调整（默认 5 分钟）
- 外观支持亮色/暗色/跟随系统，支持背景图
